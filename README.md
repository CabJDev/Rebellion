# Rebellion

## Overview
This is a game that was made for my capstone project at MacEwan University. The main goal was to figure out a way to develop a game similar to the "Jackbox Party Games". Players are able to use a web browser to join a lobby with a provided code and control the actions of their character.

This is the Unity game side of the project. One person, the host, must run this game and provide a way to show their game to all players, whether it be hooking it up to a TV, or streaming their game.
